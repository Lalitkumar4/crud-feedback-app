## Usage

#### Install dependencies

```
npm install
```

#### Run react dev server :3000

```
npm start
```

#### Run json server :5000

```
npm run server
```

#### To run both at the same time (This will run React on port :3000 and JSON-server on port :5000)

```
npm run dev
```

#### Production Build

```
npm run build
```
